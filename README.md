# School-Progress-Manager
This program that takes your schoolwork (if you are in Epic Ed) and gets your grade, pace needed and allowed goals to be made.


Extensions setting.
Go to about:debugging#/runtime/this-firefox.
Add manifest as an addon.

How to use the website.
Go to the all of gradebook in Epic Ed.
Download the .json.
Boot up Assignments Dashboard.html.
Load the .json file you downloaded.
Done! Do whatever you want and it will keep it saved.
