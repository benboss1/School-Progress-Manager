# School-Progress-Manager
  This program that takes your schoolwork (if you are in Epic Ed) and gets your grade, pace needed and allowed goals to be made.


# Extensions setting (Firefox)
  1. Go to about:debugging#/runtime/this-firefox.
  2. Add manifest (in Gradebook Exporter) as an addon
  3. You will need to reload it after you close FireFox

# Extensions setting (Chrome)
  1. Go to chrome://extensions
  2. Turn on Developer Mode
  3. Load unpacked
  4. Choose the Gradebook Exporter folder as an extension

# How to use the website.
  1. Go to the all of gradebook in Epic Ed.
  2. Download the .json.
  3. Boot up Assignments Dashboard.html.
  4. Load the .json file you downloaded.
  5. Done! Do whatever you want and it will keep it saved.

# Compatibility checks
  This is tested on a windows 11 computer on FireFox. It should work on any computer and any broswer but it might have problems. If so! Please report it to me and I will try to fix it as soon as possible.

# Notices
  This doesn't store any of this information anywhere but your computer. This is a completely local program, there is no privacy risk with this. It's basically just downloading the gradebook and doing some calculations locally, it's a pretty simple program.
